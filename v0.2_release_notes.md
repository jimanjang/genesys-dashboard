# Genesys Dashboard v0.2 Release Notes

## [2026-03-26] v0.2: Looker Studio Analysis Integration

v0.2 버전에서는 기존의 실시간 모니터링 기능에 더해, BigQuery 데이터를 기반으로 한 Looker Studio 분석 리포트 통합이 완료되었습니다.

### 주요 변경 사항

#### 1. Looker Studio 리포트 통합 (Analysis Tab)
- **BigQuery 연동 리포트**: 과거 이력 및 통계 데이터를 시각화한 Looker Studio 리포트가 대시보드 내에 직접 임베딩되었습니다.
- **실시간 전환**: 상단 탭 메뉴를 통해 '실시간 모니터링'과 '데이터 분석' 화면을 즉시 전환하며 확인할 수 있습니다.

#### 2. 사용자 인터페이스(UI) 고도화
- **탭 기반 내비게이션**: 대시보드 내에서 자연스러운 화면 전환을 위한 프리미엄 탭 스타일이 적용되었습니다.
- **전체 화면 리포트**: Looker Studio 리포트를 최적화된 해상도로 볼 수 있도록 분석 영역 레이아웃을 조정하였습니다.

### 기술 스택 업데이트
- **Frontend**: React, CSS Modules (LookerReport)
- **Integration**: Google Looker Studio Embed SDK (via iframe)

- 리포트 기간 필터링 연동 최적화
