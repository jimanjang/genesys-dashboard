# Genesys Dashboard v1.4 Release Notes

## [2026-05-18] v1.4: 페이팀/알바팀 실시간 대시보드 레이아웃 개편 및 실시간성/정확도 강화

v1.4 버전에서는 페이팀과 당근알바팀의 개별 요구사항을 반영하여 대시보드 레이아웃을 전면 개편하고, 상담원의 계정 상태 유지 시간의 실시간 연동 강화 및 알바팀 지표의 정밀 측정 로직을 개선하였습니다.

### 주요 변경 사항

#### 1. 페이팀 대시보드 레이아웃 리팩토링 및 3개 큐 연동
- **3개 전용 대기열(큐) 연동**: 일반내선(당근 페이 관련 문의), 바로구매(당근 페이_바로 구매 문의), 금융피해 신고센터(당근페이 금융피해 신고센터) 데이터를 연동하여 화면에 배치했습니다.
- **2x2 그리드 기반 메트릭 그룹 카드 개편**: 기존 3단(대기/인입/포기) 구조에서 벗어나 **인입호, 응대호, 포기호, 대기호** 4가지 지표를 한 화면에 노출하는 2x2 그리드 레이아웃으로 개편하여 가독성을 극대화했습니다.
- **리드타임 위젯 최적화**: 공간 활용도 향상을 위해 LeadTimeCard(평균 처리 시간 - AHT)의 표시 크기를 `lg`에서 `md`로 조절하였습니다.
- **전체 응대율 공식 정밀화**: 기존 일반내선 1개 큐만 기준 삼던 전체 응대율을 3개 큐 전체 합산 방식으로 변경하고 너비를 조정했습니다 (17% -> 15%).
- **계정 상태 테이블 시인성 강화**: 폰트 크기를 상향 조정하고 화면 우측 패널의 가로 폭을 확장하여 (19% -> 21%) 대시보드 시인성을 개선했습니다.

#### 2. 계정 상태 유지 시간의 완전한 실시간 동기화 (1초 단위 Ticking Timer)
- **상태 변경 타임스탬프 전송**: 백엔드 데이터 구조를 개선하여 단순히 계산된 시간 문자열 대신 상태가 바뀐 정확한 시점(`statusChangedAt` ISO 타임스탬프)을 클라이언트에 전송하도록 수정했습니다.
- **프론트엔드 실시간 클라이언트 타이머 구축**: `AgentTable` 내부에 1초 단위로 경과 시간을 자동 업데이트하는 타이머 컴포넌트를 탑재하여, 브라우저 새로고침 없이도 초 단위로 째깍째깍 움직이는 극도의 실시간성을 구현했습니다.
- **정확도 보정**: 상담원이 상태가 변경될 때마다 시점 기준 타이머가 초기화되어 실제 Genesys 운영 화면의 유지 시간과 100% 동기화되도록 보정했습니다.

#### 3. 당근알바팀 지표 설정 정밀 보정 및 큐 확장
- **'당근알바 호전환' 신규 큐 추가**: 기존 2개 큐 체제에서 **'당근알바 호전환'** 큐를 백엔드 registry 및 프론트엔드에 추가하여 3개 큐의 합산 데이터로 집계하도록 확장했습니다.
- **대기호 집계 공식 고도화**: 단순 대기 중인 콜 수 집계(`oWaiting`)에서 상담원의 전화벨이 울리는 **'알림(oAlerting)'** 상태와 통화 중인 **'인터랙션(oInteracting)'** 상태의 콜까지 포함하도록 백엔드 집계 엔진을 개선했습니다.
- **음성 채널 필터 정합성 적용**: 모든 메트릭에 대해 '음성(voice)' 미디어 유형 필터의 정합성을 확보하여 Genesys 대시보드 위젯과의 오차를 전면 제거했습니다.

#### 4. 상담원 정렬 로직 고도화 (Advanced Agent Status Sorting)
- **유지 시간 오름차순 2차 정렬**: 상담원의 상태가 동일할 경우 **유지 시간(Duration)이 짧은 순서(오름차순)로 정렬**하는 조건 추가했습니다. 활성 인터랙션을 직관적으로 모니터링하고 테이블 정렬 균형을 유지할 수 있도록 알고리즘을 고도화했습니다.

### 기술 스택 업데이트
- **Backend (NestJS)**: Genesys API Service Query (`oAlerting` metric 추가), `AgentStatus` schema 확장 (`statusChangedAt` 필드 지원), TEAM_QUEUE_CONFIG 확장
- **Frontend (React)**: AgentTable 내 ticking timer component (`AgentDuration`) 구현, RealtimeDashboard/MetricCards 레이아웃 및 2x2 그리드 CSS 개선
